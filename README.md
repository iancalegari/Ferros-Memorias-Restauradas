# Coleção de Fotos Antigas de Ferros - Restauradas com IA
Este projeto é um site simples desenvolvido em HTML, CSS e JavaScript, que exibe uma coleção de fotos históricas de Ferros, MG, restauradas utilizando a IA Gemini. O site tem como objetivo apresentar a melhoria das imagens antigas, mostrando tanto as versões originais quanto as restauradas.
## Descrição do Projeto
A coleção é composta por fotos de Ferros que foram restauradas automaticamente utilizando inteligência artificial. A IA Gemini foi utilizada para melhorar a qualidade visual das imagens, com foco em detalhes como clareza, contraste e correção de danos.

Aviso importante: Devido à natureza das imagens antigas, podem ocorrer erros durante a restauração. Detalhes como rostos, corpos e alguns objetos podem ter modificações devido a limitações do processo de IA.

## Funcionalidades do Site
- Galeria de Fotos: O site exibe fotos antigas e suas versões restauradas.
- Comparação de Imagens: É possível comparar as fotos originais com as restauradas através de uma ferramenta de deslizar.
- Design Responsivo: O layout foi projetado para ser acessível e agradável em diferentes dispositivos (desktop, tablet, celular).

## Aviso Sobre a Restauração
As imagens passaram por um processo de restauração automática com a IA Gemini. No entanto, devido à antiguidade das fotos, podem ocorrer erros de precisão, especialmente em relação a rostos, corpos e elementos contextuais. Isso ocorre devido às limitações da IA e à qualidade original das imagens.
